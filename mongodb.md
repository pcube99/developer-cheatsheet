### 1) Show available databases
```
show dbs
```
### 2) create new database
```
use "database_name"
```
### 3) create new collection
```
db.collection_name.insertOne({ name: "hello world" })
```
### 4) show all data within collection
```
db.find()
```
### 5) show all data within collection (pretty format)
```
db.find().pretty()
```
